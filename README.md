# 📊 Customer Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python**, **SQL Server**, and **Power BI**. The project covers data cleaning, SQL analysis, and interactive dashboard creation to generate meaningful business insights.

---

## 📌 Project Overview

This project demonstrates a complete data analytics workflow:

- Loaded the customer shopping dataset using Python
- Cleaned and prepared the data by handling missing values, duplicates, and formatting issues
- Imported the cleaned dataset into SQL Server
- Performed SQL analysis to answer business-related questions
- Built an interactive Power BI dashboard to visualize key insights

## 📈 Dashboard Features

The Power BI dashboard includes:

- Total Customers
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Subscription Status
- Interactive Filters (Category, Gender, Subscription, Shipping Type)
